# Sasam (สะสม)

🃏 พานักสะสมสู่ยุคใหม่! ก้าวสู่โลกแห่งการสะสมการ์ดดิจิทัลด้วย "Sasam"
หากคุณเป็นคนหนึ่งที่หลงใหลในเสน่ห์ของการสะสมการ์ด ไม่ว่าจะเป็นการ์ดเกม การ์ดไอดอล หรือการ์ดอาร์ตสุดหายาก... คุณเคยประสบปัญหาเหล่านี้ไหม?

สมุดเก็บการ์ดเต็ม ไม่มีที่จัดเก็บ

กลัวการ์ดใบโปรดเป็นรอย ลอก หรือเสียหายจากความชื้น

อยากแลกเปลี่ยนหรืออวดกรุการ์ดกับเพื่อน แต่พกพาไปไหนมาไหนลำบาก

หมดปัญหาเหล่านั้นด้วย Sasam แอปพลิเคชันที่จะเปลี่ยนประสบการณ์การสะสมการ์ดของคุณให้ง่าย สนุก และปลอดภัยกว่าที่เคย!

🌟 Sasam คืออะไร?
Sasam คือแพลตฟอร์มศูนย์รวมสำหรับนักสะสมการ์ด ยุคใหม่ ที่ออกแบบมาเพื่อรองรับการสะสม การจัดการ และการสร้างคอมมูนิตี้ในที่เดียว ไม่ว่าคุณจะเป็นนักสะสมสายไหน Sasam ก็พร้อมตอบโจทย์ด้วยระบบที่ลื่นไหลและใช้งานง่าย

🚀 ฟีเจอร์เด่นของระบบ Sasam ที่คุณไม่ควรพลาด

1. 🗂️ Digital Album (สมุดสะสมการ์ดดิจิทัลส่วนตัว)
   จัดเรียงหมวดหมู่การ์ดของคุณได้อย่างเป็นระเบียบ มองเห็นคอลเลกชันทั้งหมดได้ในหน้าเดียว ระบบช่วยบันทึกรายละเอียดของการ์ดแต่ละใบ ทั้งระดับความหายาก (Rarity) และประวัติการได้มา

2. 🔍 Easy Search & Catalog (ค้นหาและตรวจสอบคอลเลกชัน)
   มีระบบฐานข้อมูลการ์ดที่ครบถ้วน ช่วยให้คุณเช็กได้ทันทีว่าการ์ดในชุดนั้นๆ คุณเก็บครบแล้วหรือยัง เหลือใบไหนที่ยังขาดอยู่ ทำให้การเก็บสะสมมีเป้าหมายชัดเจนขึ้น

3. 🤝 Trade & Marketplace (ระบบแลกเปลี่ยนและซื้อขาย)
   ไม่ต้องเสียเวลาโพสต์หาตามกลุ่มโซเชียลให้อีนุงตุงนัง ระบบของ Sasam มีพื้นที่ให้คุณตั้งจับคู่แลกเปลี่ยนการ์ดใบที่ซ้ำกับคนอื่น หรือซื้อขายได้อย่างปลอดภัยไร้กังวล

4. 🌐 Collector Community (สังคมของคนรักการ์ด)
   แชร์สมุดสะสมการ์ดของคุณให้เพื่อนๆ หรือนักสะสมทั่วโลกได้เข้ามาชื่นชม แลกเปลี่ยนมุมมอง และอัปเดตข่าวสารวงการการ์ดได้ตลอด 24 ชั่วโมง

💡 ทำไมต้องเริ่มสะสมการ์ดกับ Sasam?
พกพาง่าย ไปได้ทุกที่: ยกกรุการ์ดทั้งหมดของคุณไว้ในสมาร์ตโฟนเครื่องเดียว

ไม่มีวันเสียหาย: การ์ดดิจิทัลจะคงสภาพสมบูรณ์แบบตลอดกาล

ปลอดภัย เชื่อถือได้: ระบบถูกออกแบบมาเพื่อปกป้องข้อมูลและคอลเลกชันอันมีค่าของคุณ

## Sasam 1.0

## ความสามารถของระบบ (Features)

ในเวอร์ชันนี้ Sasam มาพร้อมกับระบบพื้นฐานที่ช่วยให้ผู้เล่นสนุกกับการสะสมและแลกเปลี่ยนการ์ด ดังนี้:

- 🖼️ **คอลเลกชันการ์ด (Collection Hub):** จัดเก็บและแบ่งหมวดหมู่การ์ด (เช่น การ์ด 77 จังหวัด, ของกิน 7-11, ไอเทมยุค 90s) สามารถดูรายละเอียดการ์ด ความหายาก (Rarity) คำขวัญ และสเตตัสของการ์ดแต่ละใบได้
- 🎲 **ระบบสุ่มการ์ด (Gacha System):** สามารถเลือกคอลเลกชันที่ต้องการและใช้ 50 เหรียญ (Coins) ในการสุ่มซองการ์ดเพื่อลุ้นรับการ์ดระดับต่างๆ
- 🛒 **ตลาดกลาง (Marketplace Live):** ระบบซื้อ-ขาย แลกเปลี่ยนการ์ดระหว่างผู้เล่น สามารถค้นหาการ์ดที่ต้องการ และตั้งราคาขายการ์ดที่มีด้วยเหรียญในเกม
- 📍 **ภารกิจเสริม (Side Quests):** ระบบเช็คอินสถานที่ด้วย GPS (จำลองการรับการ์ดตามพื้นที่) และมินิเกมตอบคำถามทายปัญหาเพื่อรับเหรียญเพิ่มเติม
- 📊 **พอร์ตโฟลิโอ (Portfolio & Stats):** หน้าจอสรุปสถิติความคืบหน้าในการสะสมการ์ดของตนเอง เปอร์เซ็นต์ความสำเร็จ และยอดเหรียญคงเหลือ
- 📱 **ระบบเข้าสู่ระบบ (Authentication):** รองรับการเข้าใช้งานผ่าน LINE (LIFF), เข้าด้วยชื่อผู้ใช้งานทั่วไป, และโหมดผู้ทดสอบระบบ โดยเชื่อมต่อกับ Google Sheets เป็นฐานข้อมูลหลัก และมีระบบ Mock Data รองรับกรณีไม่ได้เชื่อมต่อ API
